# React Native Map App

Um aplicativo React Native que exibe um mapa usando **Google Maps**, permitindo acessar a localização do usuário em tempo real.

## Tecnologias Utilizadas

- **React Native** - Framework para desenvolvimento de aplicativos móveis.
- **Expo** - Plataforma para desenvolvimento e teste rápido de apps React Native.
- **expo-location** - Biblioteca para acessar a localização do dispositivo.
- **expo-permissions** - Para solicitar permissões de acesso à localização.

## Funcionalidades

- Exibir mapa interativo utilizando o Google Maps.
- Solicitar permissão ao usuário para acessar a localização.
- Mostrar a posição atual do usuário no mapa.
